## Features

- No Dependencies.
- Minimal API. Provides 3 hooks - `useForm`, `useFormState`, `useFieldState`.
- Blazing Fast. Allows you to re-render form inputs only if necessary.
- Tiny Size. ~2KB gzipped.

## Install

```bash
npm install --save react-form-hooks
```
